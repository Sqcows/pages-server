What is Forgejo Pages?

Forgejo Pages Server is a Traefik middleware plugin that brings static site hosting capabilities to Forgejo and Gitea, similar to GitHub Pages and GitLab Pages.

Key Features

  • Static Site Hosting: Serve HTML, CSS, JavaScript, and assets directly from your repositories
  • Automatic HTTPS: Integration with Traefik's Let's Encrypt for automatic SSL certificates
  • Custom Domains: Use your own domain names with automatic DNS configuration
  • Profile Sites: Create personal pages at username.pages.example.com
  • High Performance: Built-in caching for sub-5ms response times
  • Easy Configuration: Simple .pages file in YAML format

How It Works

  1. You create a repository in Forgejo/Gitea with a public/ folder
  2. Add a .pages file to enable hosting
  3. The plugin serves files from the public/ folder
  4. Traefik handles HTTPS and routing automatically
  5. Your site is live at the configured URL

Technology Stack

  • Traefik: Modern reverse proxy and load balancer
  • Forgejo/Gitea: Git hosting platform
  • Let's Encrypt: Free SSL certificate authority
  • Cloudflare: DNS management (optional)
  • Go: Plugin implementation language
